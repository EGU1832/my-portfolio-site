## Directory Structure
```
my-portfolio-site/
├── app/                /* App Router (main page) */
├── components/         /* UI components */        
│   ├── ProfileCat.tsx        /* 3D interactive profile */
│   ├── MarkdownRenderer.tsx  /* Custom markdown renderer */
│   :
│   └── {Component}.tsx       /* Components for Web UI */
├── public/             /* Static assets */
│   ├── models/         /* GLB 3D model */
│   ├── images/         /* UI images */
│   ├── archive-md      /* .md files */
│   └── projects/       /* Demo media */
├── README.md           /* README */
└── next.config.ts
```
